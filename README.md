# SOPTIFY-CLONE 🎵

A simple Spotify-inspired music player clone built with **HTML, CSS, and JavaScript**.  
This project replicates the core UI and basic functionality of Spotify, including:

- 🎧 Music player interface  
- 📂 Playlist and song listing  
- ▶️ Play, pause, and control buttons  
- 🎨 Responsive design with custom styling  

## 🚀 Getting Started
Clone the repository and open `index.html` in your browser to explore the app.
